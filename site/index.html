Alan's Web Component Techniques

How I'm working with web components

Introduction

The idea of working with web components appealed to me the first time I heard them. A standards based way to build isolated, reusable widgets is a very satisfying idea.

Working with web components has been a bigger learning curve than I expected. Some of that is because there doesn't seem to be a huge amount of info out there about them. This site is an effort to help that. It'll be my working-in-public collection of notes about how I use them.
